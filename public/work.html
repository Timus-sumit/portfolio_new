Bug Tracker

  • A bug tracking system or defect tracking system is a software application that keeps track of reported software bugs in software development projects.
  • Technologies/Frameworks used: React, Node, Express, MongoDB, Firebase
  • This bug tracker is made for internal use in a company/organization.
  • Database records facts about known bug such as the time a it was reported, its severity/priority, and details on how to reproduce the bug; as well as the identity of the person who reported it and any programmers who may be working on fixing it.

Neuro Fuzzy Hybrid System for Diabetes Diagnosis

  • Neuro fuzzy system is based on fuzzy system which is trained on the basis of working of neural network theory.
  • An Adaptive Neuro Fuzzy Inference System was made using Genetic Algorithm.
  • Languages/Libraries used: Python, Panda, NumPy, Scikit-learn
  • The Model was trained using the famous Pima Indian Diabetes Dataset.
  • An accuracy of 80% percent was achieved. A higher accuracy can be achieved by using a larger dataset.

Task Management API

  • Design of this Task Management is REST API based.
  • Technologies/Frameworks used : Node, MongoDB, Mongoose, Express
  • JWT Authentication is used for user login/signup.
  • It supports all CRUD operations along with searching and sorting.

Lyrics Finder Chrome Extension

  • I wanted to read lyrics of a song while looking at the music video, so I made a Chrome extension for it.
  • Language used: Javascript
  • Made using Shazam API and Genius API.
